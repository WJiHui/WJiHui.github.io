“ho:” 短格式参数
当参数选项后面不带附加参数时”:”号时，只可以表示开关状态时，比如-h
当参数选项后面不带附加参数时”:”号时，输入参数后可带参数值，比如-o output
[“help”, “output=”] ：长格式参数
“help”是一个开关选项； –help
“output=”有等号则表示后面应该带一个参数值，比如 –output args
